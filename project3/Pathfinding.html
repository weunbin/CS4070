Instructions :

Press the reset button to clear the grid. Input a number for both the height and width input boxes and then press Reset Grid to configure size of the grid.
Click once on a tile to set the start node (GREEN). Click again on a tile to set the goal node (RED). Every consecutive click after the goal node will be obstacle nodes (BLUE).
You can control the speed of the animation by sliding the animation slider to the right to go faster and left to go slower.
